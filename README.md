# CoRide
CoRide is a ride-sharing platform designed for employees to share rides,
reduce costs, and minimize environmental impact.

## Features
- Offer rides
- Book rides
- Earn points and rewards
- Rating

## Technologies
- Java
- MySQL

## Installation
1. Clone the repository
2. Open in IntelliJ
3. Run the application

## How to run in Git bash
ΠΡΙΝ απο τα παρακατω βήματα, ρυθμισε τα στοιχεια σου (ιδανικα όπως τα εχεις και στη σελιδα του github)
git config --global user.name "Marchi16"
git config --global user.email "το-email-σου@gmail.com"

1. Clone repo: git clone https://github.com/Marchi16/softwareEngineering.git
2. Cd in main: cd softwareEngineering
3. Για να δεις καθε αλλαγη που εγινε στο repository: git pull origin main
4. Οταν κανεις αλλαγή (κυρίως κώδικα): notepad arxeio.java (οπου αρχειο οποιοδηποτε αρχειο κωδικα υπάρχει στο repo)
5. Μετα τις αλλαγες: git add .
                     git commit -m "μήνυμα"
                     git push origin main

6. Για να προσθεσουμε καποιο αρχείο απο τον υπολογιστη μας: 1) Εισαγωγη μεσω ιστοσελιδας github
